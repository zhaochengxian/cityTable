# city-table

## how to use

```
yarn add @banyinbanying/city-table or npm i @banyinbanying/city-table
```

### then you add this two line in vue main.js

```
import CityTable from "../packages/index";
Vue.use(CityTable);
```

### after you can use it in component like this

```
<template>
  <div id="app">
    <cityTable></cityTable>
  </div>
</template>

<script>
export default {
  name: "App",
};
</script>

<style></style>

```
